# Zenn トピック厳選リスト

このアドベントカレンダー用に厳選したZennトピックのリストです。各トピックの記事数を基準に、プロジェクトの内容に関連性が高く、人気のあるものを選定しています。

## 選定基準
- プロジェクトの技術スタック・内容との関連性
- Zennでの記事数・人気度
- 類似トピックがある場合は記事数が多い方を優先

## 使い方

各記事には、内容に最も関連性の高いトピックを3-5個選択して付与します。

- **必須トピック**: ai, go, llm (ほぼ全記事)
- **内容に応じて**: security, agent, chatgpt, gemini, claude など
- **技術詳細**: test, api, cli, json, prompt など
